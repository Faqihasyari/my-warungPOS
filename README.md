# 🛒 My Warung POS

A modern Point of Sale (POS) web application built with **Next.js 16**, **Prisma**, and **PostgreSQL (Neon)**.

Designed for small businesses and local shops to manage products, transactions, and sales reports efficiently.

---

## 🚀 Features

### 🔐 Authentication
- Role-based access (OWNER & CASHIER)
- Secure login with NextAuth
- Session-based authorization

### 📦 Product Management
- Create, update, and delete products
- Search & pagination
- Stock tracking
- Description & packaging (pcs per dus)
- Multi-store support (Owner)

### 🛒 POS System
- Search-based product selection
- Cart system with quantity control
- Stock validation (no over-selling)
- Auto currency formatting (Rp)
- Payment input & change calculation
- Responsive modern UI

### 📊 Dashboard
- Total products
- Total transactions
- Today's revenue
- 🏆 Top 5 best-selling products
- 📜 Sales history

---

## 🧰 Tech Stack

- **Frontend:** Next.js (App Router)
- **Backend:** Next.js API Routes
- **Database:** PostgreSQL (Neon)
- **ORM:** Prisma
- **Auth:** NextAuth
- **Styling:** Custom CSS (modern POS UI)
